## Project Name: High Cloud Airlines Data Analysis

### Overview
Analyzed real-world airline operational data to find trends 
in carrier performance, route popularity, and load factors.

### Tools Used
- Excel (VLOOKUP, Pivot Tables, Power Query)
- SQL (JOINs, GROUP BY, Aggregations)
- Tableau (Interactive Dashboards)

### Key Insights
- Identified top 10 carriers by passenger preference
- Weekend load factor was 18% higher than weekdays
- Top 5 routes accounted for 40% of total flights
