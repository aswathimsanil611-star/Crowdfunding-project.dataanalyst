# Kickstarter Crowdfunding Data Analysis

## Overview
Analyzed a real-world Kickstarter crowdfunding dataset to extract key business 
insights, track campaign success metrics, and create interactive dashboards to 
support data-driven decision-making.

---

## Tools Used
- *Excel* – Data cleaning, Power Query, calendar table creation
- *SQL* – Data extraction, aggregations, filtering
- *Power BI* – DAX, data modeling, KPI dashboards
- *Tableau* – Interactive visualizations and filters

---

## Key Objectives
- Identify patterns in successful crowdfunding campaigns
- Analyze funding amounts, backer counts, and campaign durations
- Track project performance by category, location, and time period
- Calculate success percentages by category, year, and goal range

---

## What I Did
- Converted epoch timestamp fields into readable date formats
- Built a calendar table with financial month, quarter, and year-month 
  attributes using Power Query and DAX
- Developed a multi-table relational data model in Power BI
- Created KPIs to measure project counts by category, location, and time
- Analyzed successful campaigns to find patterns in funding and backers
- Delivered interactive dashboards for business decision-making

---

## Key Insights
- Technology and Design categories had the highest success rates
- Campaigns with shorter durations (20–30 days) performed better
- Higher backer count strongly correlated with campaign success
- Funding goal range of $1K–$10K had the most successful outcomes
